# Sensor de luminosidade

<p align='center'> Implementação de um sensor de luminosidade utilizando o sensor BH1750. </p>

## Pinos utilizados 

Display LCD 240x320:

````
+-------------------------+
| VCC  --------   (VCC)   |
+-------------------------+
| GND  --------   (GND)   |
+-------------------------+
| SCL  -------- (GPIO 18) |
+-------------------------+
| SDA  -------- (GPIO 19) |
+-------------------------+
| RES  -------- (GPIO 20) |
+-------------------------+
| CS   -------- (GPIO 17) |
+-------------------------+
| DC   -------- (GPIO 16) |
+-------------------------+
````

Sensor de luminosidade BH1750:

````
+-------------------------+
| VCC  --------   (VCC)   |
+-------------------------+
| GND  --------   (GND)   |
+-------------------------+
| SCL  --------  (GPIO 1) |
+-------------------------+
| SDA  --------  (GPIO 0) |
+-------------------------+
````

Servo motor SG90:

````
+-------------------------+
| VCC  --------   (VCC)   |
+-------------------------+
| GND  --------   (GND)   |
+-------------------------+
| SDA  --------  (GPIO 2) |
+-------------------------+
````
